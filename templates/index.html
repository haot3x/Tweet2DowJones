Idea

With the increasing popularity of the social network, collecting public mood become possible via natural language analysis on Twitter tweets.

The goal of this project is to develop a system to track and analyze Dow Jones by modelling the correlation between stock market indicator and public social media mood trend.

The tweets used were collected via Twitter search APIs with several key words, and then processed with a sentiment analyser provided here. Dow Jones data was collected from Yahoo Finance.

Input Stats

We used twitter api to grab public tweets and Yahoo Finance to grab Dow Jones Index value. During the period of April 7th 2014 ~ April 18th 2014, the application has collected 234,239 tweets and 1,783 Dow Jones values in total.

Conclusion

Cross Correlation

The cross correlation graph is shown on the left, and assume we have n1 and n2 array aligning on the x-axis
>> [acor,lags]=xcorr(n1,n2);
>> stem(lags,acor/length(n1));
 the result shows we have many outstanding peaks, which means we have strong correlations between tweets emotion and Dow Jones index on those particular time periods.

Summary

Public mood trend can be simulated on certain degree via sentiment analysis on social media data. When the emotions on twitter have more positive points, Dow Jones index will have more fluctuations in the near future. Emotional outbursts on Twitter produces a predictor of how the stock market will perform.
